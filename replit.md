# Math Quest - Fun Math Game for Kids

## Overview
A colorful, browser-based math game for kids that covers addition, subtraction, multiplication, and division. Players answer math questions, get instant feedback, and track their score and streaks.

## Recent Changes
- 2026-02-20: Initial build of Math Quest game with all 4 operations, score tracking, streaks, encouraging messages, and kid-friendly colorful theme

## Project Architecture
- **Frontend**: React + TypeScript with Tailwind CSS and shadcn/ui components
- **Game Page**: `client/src/pages/math-game.tsx` - Main game component with all logic
- **Theme**: Purple/pink kid-friendly color scheme defined in `client/src/index.css`
- **Font**: Poppins for friendly, rounded text
- **Animations**: Framer Motion for smooth transitions

## Key Features
- 4 math operations: Addition, Subtraction, Multiplication, Division
- Score and streak tracking
- Encouraging feedback messages
- Auto-generates new question after answering
- Enter key support for quick submission
- Responsive design for all screen sizes

## User Preferences
- Code should be simple and well-commented (kid-friendly learning project)
